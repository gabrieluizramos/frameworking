Usage

To use the lib, just import de plugins.extended.js script into your project and call the $.init function.
This function is used passing at least 4 arguments. In order, they are:

  1. Effect name ( 'dropdown', 'slider', 'tabs', 'modal', 'accordion' );
  2. Event that will listen to execute the effect ('click', 'mouseenter', 'mouseenter mouseleave' and all the events used in jquery);
  3. Element that will trigger the execution (I prefer using 'data-' elements, simples as [data-accordion="header"] to accordion's header, but you can use jquery notation as you like (classes and id);
  4. Element that will receive the execution, such as [data-accordion="panel"] to execute the accordion effect in the elements that have this attribute ( like example above );

You can aways pass a 5th optional argument that will be used as a time to execute the effect. Your javascript function will sound just like this: $.init( 'dropdown','click','[data-dropdown="link"]','[data-dropdown="content"]', 500 );

You can explore the plugins.extended.js lib to other functions, animations and customize your effects.
All the css used only in elements of the extended lib is setted up by jquery too.
When initialized, the functions will display a message in your console log containing the name and the activation status ( true to success and false to fail ).
You can call the functions without the $.init effect, just calling $.functionName and passing the required arguments.

Animations List

  • dropdown: execute a dropdown effect, just like reponsive menus;
  • slider: performs a simple slide ( up / down );
  • tabs: execute a tab animation;
  • modal: execute a modal animation.
    Important: To completely use this effect is necessary to create other two elements inside your modal "content", and they are elements with attributes [data-modal="close"] that, with a "X" inside will perform the close button, and [data-modal="outer"], that will perform the close action when user clicks ( or execute the event you passed to the function ) in the outside of your modal full ( black ) panel;
  • accordion: performs an accordion simple animation.
  • parallax: execute a parallax background simple animation.
    Important: To completely use this effect, just call 'parallax', pass the element that will have the background parallaxed, and the [data-*] attribute with the animation speed value ( just set something like [data-parallax="5"] and pass '[data-parallax]' as the 4th argument, you can just an [data-parallax] attribute with the value, and pass as argument only one time. without passing the 4th argument, the function will assume that the 2nd argument is the data-attribute and use its value to proceed with the animation ).

About

This project is in search of execute the most used front-end animation using jquery, without import plugins or other heavy scripts that make your pages load slower.
FrameWorking lib is in constant update.
When a update is ready to use, I will pushing the files to the repo.

Hope you enjoy it!
